## 当前结论边界

v2 的角色是：

```text
无实测数据、面向真实仪器约束的模拟补强
```

它补充原来的工程逻辑和全量模拟结果，使相对模拟判断更可信；它不等同于真实采集、实测校准或实验验证。

允许的结论（双口径共享）：

- 相对模拟证据；
- 受限仪器情景下的相对先验判断；
- 锁定路线在合成模型内的稳定性；
- 近壁网格、路线角色和窄通道风险先验的审计结论；
- P0 相对审计和 P6/P8/P10/P12/P14/P16 trace-only 排名不稳定结论；
- selected-annulus lens 在 Phase 2 / Phase 2.5–2.11 内的 B1 anchor target-fitting / reproduction lens score（最佳 single global response compression `gamma ≈ 0.749`，score `2.033`）；
- 口径 B B2 EV+gold full-grid 当前 3seed 10000e 结论：`fixed_660_gold` selected-annulus recommendation family 转向 `404 nm / W500` mid-depth；`per_wavelength_gold` selected-annulus recommendation family 保留 `660 nm / W800` D1400-D1500，尤其 sharp MSC/sEV prior；B6/B7 单 seed 1000e 仅作历史方法背景；`488/532` 只作 trend/control，gold rows 只作 anchor / Tsuyama diagnostics；
- shared-dual normalization 下 physical stochastic event samples 只按物理事件计数，不能把两个 analysis views 双倍计入；
- ET-2030 + LI5640 instrument-aware feasibility 量级估计（current input / TIA comfortable，50 Ω voltage path 多数 below sensitivity）；
- 未来要进入实测或校准前必须补齐的 artifact 清单。

禁止的结论（双口径共享）：

- 校准信噪比；
- 校准事件概率；
- 绝对检出限；
- 真实 EV 浓度；
- 生物特异性；
- 实测空白样本安全性；
- 路线直接升级（口径 A）/ 已签发 accepted paper-calibrated candidate（口径 B）；
- 把口径 B 的 B1 anchor diagnostic geometry 写成 B2 EV 推荐尺寸；
- 把 488/532 raw metric 高分或 top1 写成最终推荐波长；
- main-660 重新定义；
- **selected-annulus 替代 all-crossing 主排序，反之 all-crossing 替代 selected-annulus / 口径 B EV application 结论**（双向并立，互不替代）；
- selected-annulus 边界变更（`0.5-0.8` 固定）；
- 把 P6-P16 trace 排名当作路线升级或单一冠军结论；
- 把 estimated-parameter reproduction lens 当作 raw physical calibration（口径 B）；
- 把 gamma response compression 当作真实物理定律（口径 B）；
- 把 `fixed_660_gold` 与 `per_wavelength_gold` 写成两次独立物理事件 campaign；
- 把 selected-annulus 写成 optical BFP annulus；
- classification accuracy 已本地复现（本地 sklearn 不可用，仍 `no_accuracy_claim`）。
