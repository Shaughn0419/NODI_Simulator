## 当前事实

- 旧 `43` 中列出的 Tsuyama 对齐 / claim gate 修正项已经完成，并已归档到 `archive/tsuyama/59_*`。
- 根目录 `43` 现在冻结为 v5 EV/NODI 设计决策层路线图；不依赖实测数据的 P0-hard / minimum schema / smoke validation / recompute manifest / P1 diagnostic-scaffold 修改已完成到可重算状态。
- 旧全量库 `results/fine_full_range_biomimetic_exosome_10000e_*` 已删除，当前正式重算目标为 `results/ev_design_full_range_biomimetic_exosome_with_anchors_10000e_*`。
- 若重算目标是 EV optical-model uncertainty，应改用 `ev_design_biomimetic_ensemble_with_anchors` profile；它在同一 `ev_design` grid 上是 `41756` cases。
- Relative design recommendation 与 absolute/global green claim 已分层；detector operator disagreement 输出 caution，不再直接阻断 relative ranking。
- Measured/calibrated BFP ROI mask 现在会进入 Tsuyama 1D projected ROI 积分；synthetic/template mask 仍只做 contract guard。
- Tsuyama selected detector-mode 现在是并行分析口径：all-crossing `detection_rate` 仍是工程 gate / 主评分输入，selected-candidate 与 edge-norm `0.5-0.8` annulus 条件率用于解释有效光学区语义；EV targeted panel 与全量 size-weighted route analysis 会额外生成 selected-annulus ranking / comparison。旧 CSV 需重跑后才会带出新增列；若直接用于新比较工具，会被标记为 selected-annulus lens unavailable/null/NaN，不会伪造 selected 结果。
- Tsuyama 几何来源已重新分层：2022 NODI 原文为 `800 / 1200 nm` 宽、`550 nm` 深，覆盖 `488 / 532 / 660 nm` 测量/比较；2024 paired POD+NODI 为 `800–1200 nm` 宽、`550 nm` 深并使用 `660 nm` probe；`800x710 nm` 只来自 2020 POD counting paper，当前只能作为 POD bridge / sensitivity case。
- Tsuyama 2022 Supplementary Table S1 fixed-index Au/Ag profile 现在只作为 paper-claim audit path：它改善 660 nm Ag/Au mean-peak ratio，但 selected-annulus detection/readout 仍不能当成 Fig. 5 / Table S1 signal-ratio 或 2022/2024 SVM classification 的数值复现。
- `tools/tsuyama_selected_annulus_joint_fit.py` 是新的 selected-annulus Tsuyama paper-fit lane：schema v2 同时评分 `800x550` 与 `1200x550` paper geometry 下的 selected-annulus detection rate、Ag/Au peak ratio、Au size scaling 和 Au30/Au20 SNR ratio；`signal_transfer_fit` variants 显式应用 bounded paper-fit silver transfer gain，`signal_size_transfer_fit` variants 额外应用 bounded Au power-law size-response correction。`joint_fit_score` 是 lower-is-better loss-style penalty，当前粒径诊断口径为 Au `20/30/40/60 nm`、Ag `40/60 nm`。当前 10000-event v2 best profile 是 `baseline_current_estimates__paper_5sigma_signal_size_transfer_fit`：raw Au size exponent `3.263` 通过 delta `-0.963` 校准到论文目标 `2.300`，无 guardrail penalty，同时保留 raw exponent 作为未校准诊断；target metadata 与 selected-annulus claim compatibility 已由 `design_claim_governance.py` 校验。
- all-crossing 不对齐 paper detection-rate target；selected-annulus paper-audit lane 当前使用 Au20 `0.30`、Au30 `0.60`、Au40 `0.78`、Au60 `0.92` 这组 2022 NODI surrogate target。2020 POD thermal counting 的近 100% Au20 结论不是 NODI calibration anchor。paper-audit lane 是 JOINT_CASES × Au `20/30/40/60` + Ag `40/60` 小网格；工程主库 lane 是 EV full-grid 推广，两者不能混写。
- `tools/tsuyama_2022_classification_lane.py` 是独立的 2022 linked 488/532 classification feature/protocol lane：它导出 Au40/Au60/Ag40/Ag60 linked event features，并已按 488 detected pulse window 取 532 maximum；当前 `400` events/class v2 smoke 为 `1600` rows、`849` paper-SVM-usable rows。由于项目仍不依赖 optional `scikit-learn`，SVM accuracy 继续写 `no_accuracy_claim`，不把 selected-annulus scalar summary 伪装成 SVM accuracy。
- `results/tsuyama_selected_annulus_pre_fullgrid_ev_robustness_20260501/` 是 full-grid 前 targeted EV 稳健性复核：`3000` events/seed、seeds `42 / 314 / 2718`、`8` workers；all-crossing 与 selected-annulus top1 在四个 EV size priors 中均稳定，selected-annulus top1 为 `488 nm / 600x1500 nm`。
- 当前全量重算执行入口默认使用 `standard` artifact profile，并包含 summary-only 流式累计、trajectory context、worker-local invariant cache 与 scalar/off 事件循环；默认 `event_block_size=32`、`event_block_rng_order="event_loop_order"`。`event_block_v3` 只保留为显式性能实验/回归对照，不改变 claim level 或结果语义。
- `block_lane_order` RNG 顺序和彻底向量化 event loop 仍只属于实验分支候选；当前对照数据不支持替换默认路径。
- 当前仍未完成的任务分两类：正式全量重算、未来接入 P2/P3 实测/高保真模型后的定量闭环。
- `archive/` 下文档是历史快照；除非追溯 provenance，不要用它们覆盖当前说明。

- 若 manifest 的 `calibration_kind` 与当前 lane 不匹配，不得把该表当作有效实测标定。
