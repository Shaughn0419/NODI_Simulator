# EV/NODI P1 Transport Residence-Time Diagnostic Contract

Date: 2026-05-10

Status:

```text
no-solver rank diagnostic; no transport or residence-time simulation
```

This document narrows the fourth P1 lane into a transport / residence-time
diagnostic contract. It remains a surrogate-risk reduction artifact. It does
not change the P0 release conclusion.

## Boundary

The lane may later compare rank evidence from bounded transport or
residence-time diagnostics against the P0 scalar, noise, EV/sample, and
coincidence proxy evidence. It may not turn those diagnostics into EV
concentration, sample count, absolute event probability, calibration,
biological specificity, or detector prediction evidence.

Required declarations:

```text
calibrated_claim_allowed = false
p0_release_conclusion_changed = false
physical_ceiling_role = surrogate_risk_reduction_only
```

SNR remains blocked. LOD remains blocked. EV concentration remains blocked.
Biological specificity remains blocked. Detector-voltage prediction remains
blocked. Absolute event probability remains blocked.

## Inputs

Required P0 sources:

| source | role |
|---|---|
| `results/post_v2_mandatory_audit/top_candidate_mandatory_audit.csv` | route geometry, comparison strata, existing rank evidence, noise, EV/sample, and coincidence relative proxy columns |
| `results/post_v2_mandatory_audit/top_candidate_particle_panel_audit.csv` | particle-panel and EV/sample profile context |
| `results/post_v2_mandatory_audit/ev_prior_contaminant_summary.csv` | EV/sample and contaminant relative policy context |

Future transport perturbation manifests, residence-time proxy manifests, or
coincidence proxy sensitivity tables require separate authorization. They are
not generated by this contract stage.

Forbidden inputs:

```text

sample-count calibration curves
absolute flow-rate calibration as event probability
route-specific manual multipliers
raw residence time as a final gate
```


The next allowed step is a stub validator or generator that emits only an empty
schema manifest. It is not a transport/residence-time implementation.
