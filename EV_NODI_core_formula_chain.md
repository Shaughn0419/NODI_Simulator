# EV/NODI Core Formula Chain

**Project:** EV/NODI engineering simulator for relative channel/wavelength selection in nanofluidic interferometric scattering detection.

---

## Executive summary

The model can be read as the following forward chain:

```text
input parameters
→ optical/material properties
→ intrinsic particle scattering by Mie or core-shell Mie
→ finite collection / detector operator
→ nanochannel diffraction reference field
→ event trajectory and illumination envelope
→ time-dependent detector-mode scattering field
→ interferometric NODI signal
→ readout / noise / thresholding
→ event QC and batch statistics
→ EV ensemble aggregation
→ engineering gates and design recommendation
```

The core physical idea is that the detector does not observe only the particle scattering intensity. It observes a coherent combination, in the selected detector mode, between the empty-channel diffraction/reference field and the particle-induced optical field. After blank-channel subtraction, the working detector-mode signal is:

\[
\Delta I(t)
=
|E_{\mathrm{sca}}(t)|^2
+
2\operatorname{Re}\left[E_{\mathrm{ref}}E_{\mathrm{sca}}^*(t)\right].
\]

For weak EV-like scatterers, the pure scattering term can be too small, while the cross term can provide the useful NODI-like interferometric enhancement. This interpretation is valid only under the current detector-mode surrogate and calibration limits described below.

---

## 0. Overall detector forward model

The most general detector-side coherent-field model is:

\[
I_{\mathrm{det}}(t)
=
\left\|
\mathcal{P}\left[E_{\mathrm{ch}}+E_{\mathrm{p}}(t)\right]
\right\|^2,
\]

where:

- \(E_{\mathrm{ch}}\) is the optical field generated by the empty nanochannel;
- \(E_{\mathrm{p}}(t)\) is the particle-induced optical field perturbation; in the main NODI chain this is represented primarily by the particle scattering field;
- \(\mathcal{P}\) is a detector-mode optical operator, including objective pupil, BFP mapping, slit/pinhole or ROI selection, filters, polarization projection, and detector-mode weighting;
- \(I_{\mathrm{det}}\) is the intensity in the selected detector mode or ROI.

For an empty-channel blank:

\[
I_{\mathrm{blank}}
=
\left\|\mathcal{P}E_{\mathrm{ch}}\right\|^2.
\]

After blank subtraction:

\[
\Delta I(t)=I_{\mathrm{det}}(t)-I_{\mathrm{blank}},
\]

\[
\Delta I(t)
=
\left\|\mathcal{P}E_{\mathrm{p}}(t)\right\|^2
+
2\operatorname{Re}
\left\langle
\mathcal{P}E_{\mathrm{ch}},
\mathcal{P}E_{\mathrm{p}}(t)
\right\rangle.
\]

In the current engineering implementation, the detector-mode fields are collapsed as:

\[
\mathcal{P}E_{\mathrm{ch}}\rightarrow E_{\mathrm{ref}},
\qquad
\mathcal{P}E_{\mathrm{p}}(t)\rightarrow E_{\mathrm{sca}}(t).
\]

Therefore, the working scalar/joint-overlap expression is:

\[
\Delta I(t)
=
|E_{\mathrm{sca}}(t)|^2
+
2\operatorname{Re}\left[E_{\mathrm{ref}}E_{\mathrm{sca}}^*(t)\right].
\]

This is the root equation of the current NODI-like enhancement chain. It is a detector-mode surrogate, not a calibrated full-vector detector model.

## 1. Input layer: geometry, optics, material, particle, readout, and flow

A general simulation case can be represented as:

\[
\Theta=
\{W,H,L_{\mathrm{ch}},\lambda_0,
\mathrm{NA}_{\mathrm{illum}},\mathrm{NA}_{\mathrm{coll}},
P_{\mathrm{probe}},P_{\mathrm{exc}},
n_m(\lambda),n_g(\lambda),\tilde n_p(\lambda),a,
\mathrm{readout},\mathrm{flow},\mathrm{noise}\}.
\]

Key variables:

- \(W\): nanochannel width;
- \(H\): nanochannel depth;
- \(L_{\mathrm{ch}}\): channel length or effective transit length;
- \(\lambda_0\): vacuum wavelength;
- \(n_m\): refractive index of the surrounding medium, such as water/PBS;
- \(n_g\): refractive index of the glass wall;
- \(\tilde n_p=n_p+i\kappa_p\): complex refractive index of the particle;
- \(a\): particle radius;
- \(\mathrm{NA}_{\mathrm{illum}}\): illumination numerical aperture;
- \(\mathrm{NA}_{\mathrm{coll}}\): collection numerical aperture;
- readout parameters: lock-in surrogate, bandpass/envelope response, threshold rule, peak width, and peak interval;
- flow parameters: mean velocity, velocity profile, diffusion, reflecting boundaries, and initial cross-sectional distribution.

Important current-run qualifications:

- The current run is a **fixed-velocity** design run, not a measured pressure-flow prediction.
- Probe/excitation powers are not calibrated detector-unit inputs in the current NODI-only design route.
- The POD/photothermal branch is not a quantitative active route.
- The output supports relative engineering ranking and experiment planning, not absolute detector voltage, absolute SNR, absolute LOD, or concentration inference.

## 2. Material layer: complex refractive index and medium wave number

For each wavelength \(\lambda_0\), the material layer supplies:

\[
\tilde n_p(\lambda_0)=n_p(\lambda_0)+i\kappa_p(\lambda_0),
\]

\[
n_m(\lambda_0),\qquad n_g(\lambda_0).
\]

The wave number in the surrounding medium is:

\[
k_m=\frac{2\pi n_m}{\lambda_0}.
\]

The relative refractive index used by Mie theory is:

\[
m=\frac{\tilde n_p}{n_m}.
\]

The size parameter is:

\[
x=k_m a=\frac{2\pi n_m a}{\lambda_0}.
\]

This layer converts gold, silver, polystyrene, silica, EV/sEV core-shell structures, medium dispersion, and wall dispersion into optical inputs usable by the scattering engine.

For EV/sEV cases, the safer interpretation is not a single uniform sphere with one fixed refractive index. The current engineering route uses an EV-like core-shell/ensemble surrogate because EV optical response depends on size, morphology, shell/core composition, hydration, and refractive-index heterogeneity.

## 3. Intrinsic particle scattering layer: Mie and core-shell Mie

### 3.1 Homogeneous-sphere Mie coefficients

For a homogeneous spherical particle, using the standard Bohren-Huffman-style relative-index convention, the Mie coefficients are:

\[
a_n=
\frac{
m\psi_n(mx)\psi_n'(x)-\psi_n(x)\psi_n'(mx)
}{
m\psi_n(mx)\xi_n'(x)-\xi_n(x)\psi_n'(mx)
},
\]

\[
b_n=
\frac{
\psi_n(mx)\psi_n'(x)-m\psi_n(x)\psi_n'(mx)
}{
\psi_n(mx)\xi_n'(x)-m\xi_n(x)\psi_n'(mx)
}.
\]

Here:

- \(\psi_n\) is the Riccati-Bessel function;
- \(\xi_n\) is the Riccati-Hankel function;
- \(a_n\) and \(b_n\) are the electric and magnetic multipole coefficients.

For a core-shell EV-like particle, the model uses a concentric-sphere extension. The final scattering engine still returns equivalent quantities such as:

\[
C_{\mathrm{sca}},\quad C_{\mathrm{ext}},\quad C_{\mathrm{abs}},\quad S_1(\theta),\quad S_2(\theta).
\]

### 3.2 Efficiency factors and cross sections

The scattering efficiency is:

\[
Q_{\mathrm{sca}}
=
\frac{2}{x^2}
\sum_{n=1}^{n_{\max}}
(2n+1)(|a_n|^2+|b_n|^2).
\]

The extinction efficiency is:

\[
Q_{\mathrm{ext}}
=
\frac{2}{x^2}
\sum_{n=1}^{n_{\max}}
(2n+1)\operatorname{Re}(a_n+b_n).
\]

The absorption efficiency is:

\[
Q_{\mathrm{abs}}=Q_{\mathrm{ext}}-Q_{\mathrm{sca}}.
\]

The corresponding cross sections are:

\[
C_{\mathrm{sca}}=Q_{\mathrm{sca}}\pi a^2,
\]

\[
C_{\mathrm{ext}}=Q_{\mathrm{ext}}\pi a^2,
\]

\[
C_{\mathrm{abs}}=C_{\mathrm{ext}}-C_{\mathrm{sca}}.
\]

### 3.3 Angular scattering amplitudes

The complex angular scattering amplitudes are:

\[
S_1(\theta)
=
\sum_{n=1}^{n_{\max}}
\frac{2n+1}{n(n+1)}
\left[a_n\pi_n(\cos\theta)+b_n\tau_n(\cos\theta)\right],
\]

\[
S_2(\theta)
=
\sum_{n=1}^{n_{\max}}
\frac{2n+1}{n(n+1)}
\left[a_n\tau_n(\cos\theta)+b_n\pi_n(\cos\theta)\right].
\]

The amplitudes \(S_1\) and \(S_2\) are complex. Their phase must be preserved for an interference-based model.

Under the classical dimensionless amplitude convention, the unpolarized differential scattering cross section is:

\[
\frac{dC_{\mathrm{sca}}}{d\Omega}
=
\frac{|S_1|^2+|S_2|^2}{2k_m^2}.
\]

If a software library uses a different amplitude normalization, the conversion from \(S_1/S_2\) to physical power must follow that library's convention. The current engineering run stores and uses relative detector-mode fields, so this layer supports relative comparison rather than photon-unit detector power.

For a detector-mode projection approximately parallel to the reference polarization, a useful engineering field proxy is:

\[
E_{\mathrm{sca,unit}}(\theta)
\propto
\frac{S_2(\theta)}{k_m},
\]

or a weighted polarization combination of \(S_1\) and \(S_2\). This is a field proxy, not a calibrated detector-unit field.

## 4. Detection angle and finite-aperture collection

The intrinsic scattering layer gives the full-space angular pattern, but the detector collects only:

\[
\Omega_{\mathrm{det}}\subset 4\pi.
\]

In the engineering model, the channel width provides a diffraction-angle scale:

\[
\lambda_{\mathrm{eff}}=\frac{\lambda_0}{n_m},
\]

\[
\theta_{\mathrm{base}}
\approx
\arcsin\left(m_{\mathrm{ord}}\frac{\lambda_{\mathrm{eff}}}{W}\right).
\]

Here \(m_{\mathrm{ord}}\) is an effective diffraction-order/lobe index. This is only an engineering guide. The expression is meaningful only when the argument lies within the propagating and collectable angular range. In the current run, edge cases are handled by surrogate NA/guardrail logic rather than by a full phase-filter detector solution.

### 4.1 Collection operator

Define the angular detector kernel:

\[
L_{\mathrm{det}}(\theta,\phi)
=
K_\theta(\theta)K_\phi(\phi)
P_{\mathrm{pupil}}(\theta)
T_{\mathrm{slit}}(\theta,\phi)
T_{\mathrm{pinhole}}(\theta,\phi)
\Pi_{\mathrm{pol}}(\theta,\phi).
\]

A coherent detector-mode scattering field can be represented as:

\[
E_{\mathrm{sca,det}}
=
\iint
L_{\mathrm{det}}(\theta,\phi)
E_{\mathrm{sca}}(\theta,\phi)
\,d\Omega.
\]

This expression should be interpreted as a detector-mode overlap surrogate. If the detector is treated as an incoherent pixel/ROI intensity integrator, the physically correct object is a power or intensity integral, not a simple coherent field sum over all angles.

The angular measure is:

\[
d\Omega=\sin\theta\,d\theta\,d\phi.
\]

If direction-cosine BFP coordinates are used,

\[
u=\sin\theta\cos\phi,
\qquad
v=\sin\theta\sin\phi,
\]

then:

\[
d\Omega=\frac{du\,dv}{\sqrt{1-u^2-v^2}}.
\]

Therefore, if the model moves from a \((\theta,\phi)\) surrogate to a calibrated BFP-coordinate model, the BFP-to-angle Jacobian must be handled explicitly.

### 4.2 Relative baseline normalization

Without detector calibration, the detector-mode scattering field is normalized against a baseline:

\[
E_{\mathrm{sca,norm}}
=
\frac{E_{\mathrm{sca,det}}}{E_{\mathrm{sca,baseline}}}.
\]

This absorbs unknown proportionality constants and supports relative ranking. It does **not** produce watts, photons, detector volts, or absolute SNR.

## 5. Nanochannel diffraction reference field

The NODI reference field comes from optical diffraction by the empty nanochannel, not from an external interferometer arm.

### 5.1 Thin phase-filter view of the nanochannel

The incident Gaussian field at the channel plane can be written as:

\[
f(x',y')
=A
\exp\left[-\frac{x'^2+y'^2}{\omega_z^2}\right]
\exp[-i\Phi(x',y';z_1)].
\]

A straight channel can be approximated as a thin phase filter:

\[
t(x',y')
=
\begin{cases}
e^{i\theta}, & x_c-l\le x'\le x_c+l,\\
1, & \mathrm{elsewhere}.
\end{cases}
\]

Here \(x_c\) is the channel-center coordinate. It is deliberately not denoted by \(a\), because \(a\) is already used for the particle radius.

The signed phase delay is:

\[
\theta=\frac{2\pi}{\lambda_0}(n_s-n_g)d,
\]

where:

- \(2l=W\) is the channel width;
- \(d=H\) is the channel depth;
- \(n_s\) is the refractive index of the solution inside the channel;
- \(n_g\) is the refractive index of the glass wall.

The sign of \(\theta\) depends on the time-harmonic and propagation convention. Therefore, quantitative phase or pulse-polarity claims require experimental phase calibration.

The BFP field after the lens can be represented as a Fourier transform of the channel-plane field:

\[
g(q_x,q_y)=\mathcal{F}\left[f(x',y')t(x',y')\right].
\]

A useful decomposition is:

\[
E_{\mathrm{diff,BFP}}
=
E_{\mathrm{total,channel,BFP}}-E_{\mathrm{no\,channel,BFP}}.
\]

The perturbation term scales as:

\[
E_{\mathrm{diff,BFP}}\propto(e^{i\theta}-1)u(q_x,l),
\]

where \(u\) is the width-dependent Fourier/phase-filter term.

Since:

\[
e^{i\theta}-1=2i e^{i\theta/2}\sin(\theta/2),
\]

and in the small-phase limit:

\[
e^{i\theta}-1\approx i\theta,
\]

the diffraction reference field carries an important phase component. This is why reference phase calibration is needed before any quantitative phase or polarity claim.

### 5.2 Current engineering reference route

The current fast-ranking route is:

\[
E_{\mathrm{ref}}=A_{\mathrm{ref}}e^{i\phi_{\mathrm{ref}}}.
\]

The reference amplitude is treated as a wavelength/channel-dependent engineering surrogate:

\[
A_{\mathrm{ref}}\sim \rho\,g(W,H,\lambda),
\]

where \(g(W,H,\lambda)\) represents the combined channel diffraction strength, phase-grating response, NA edge behavior, width/depth effects, and reference saturation heuristic. The configured \(\rho\) is a relative engineering scale factor, not a measured physical field ratio.

This route is valid for relative engineering ranking, but it is not a measured reference-field amplitude.

### 5.3 Future calibrated reference route

After blank-channel BFP images, slit scans, or standard-particle phase calibration are available, the route can be upgraded to:

\[
E_{\mathrm{ref}}=E_{\mathrm{ref,meas}}(W,H,\lambda,\mathrm{ROI}).
\]

Only then can the model begin to support stronger detector-unit and phase-sensitive claims.

## 6. Detector forward model: scalar collapse versus ROI mode overlap

A strict detector model should not reduce the detector to a single scalar before checking detector-mode overlap. A more general ROI expression is:

\[
\Delta I(t)
=
\iint_{\mathrm{ROI}}
\left[
|E_{\mathrm{ref}}(u,v)+E_{\mathrm{sca}}(u,v,t)|^2
-|E_{\mathrm{ref}}(u,v)|^2
\right]
J(u,v)\,du\,dv.
\]

Expanding gives:

\[
\Delta I(t)
=
\iint_{\mathrm{ROI}}
|E_{\mathrm{sca}}(u,v,t)|^2J(u,v)\,du\,dv
+
2\operatorname{Re}
\iint_{\mathrm{ROI}}
E_{\mathrm{ref}}(u,v)E_{\mathrm{sca}}^*(u,v,t)J(u,v)\,du\,dv.
\]

The current project approximates this detector physics using a shared collection-operator scalar mode and a joint-overlap coherent surrogate. This is stronger than a pure single-angle scalar model, but it remains uncalibrated and detector-resolved only in a surrogate sense.

Current interpretation:

```text
valid: relative ranking, mechanism explanation, candidate prioritization
blocked: absolute detector volts, absolute photon-unit SNR, absolute LOD, quantitative phase/polarity claim
```

## 7. Particle motion and illumination envelope

### 7.1 Flow

For a particle moving through the detection region:

\[
\mathbf{r}(t)=(x(t),y(t),z(t)).
\]

The streamwise motion should generally be written as:

\[
\frac{dy}{dt}=v_y(x(t),z(t)),
\]

or:

\[
y(t)=y_0+\int_0^t v_y(x(\tau),z(\tau))\,d\tau.
\]

The simpler expression

\[
y(t)\approx y_0+v_y(x_0,z_0)t
\]

is valid only when the transverse position is approximately constant or when \(v_y\) is intentionally frozen for a surrogate event.

### 7.2 Brownian diffusion

For diffusive motion with an effective scalar diffusion coefficient:

\[
d\mathbf{r}=\mathbf{v}(\mathbf{r})dt+\sqrt{2D}\,d\mathbf{W}_t,
\]

where \(D\) is the diffusion coefficient and \(d\mathbf{W}_t\) is the Wiener increment.

A Stokes-Einstein estimate is:

\[
D_0=\frac{k_BT}{6\pi\eta a}.
\]

Near walls, diffusion can be reduced by a hindrance factor:

\[
D_{\mathrm{eff}}=D_0 f_{\mathrm{hindrance}}(a/h),
\]

where \(h\) is the distance to the wall or a representative confinement scale. In a strict hydrodynamic treatment, diffusion can be anisotropic and position-dependent; the current implementation should be read as a near-wall surrogate with reflecting boundaries, not a full hydrodynamic Brownian dynamics solver.

### 7.3 Illumination envelope

A Gaussian **field-amplitude** illumination envelope can be written as:

\[
A_{\mathrm{env}}(t)
=
\exp\left[
-\frac{(x(t)-x_f)^2}{w_x^2}
-\frac{(y(t)-y_f)^2}{w_y^2}
-\frac{(z(t)-z_f)^2}{w_z^2}
\right].
\]

Here \(w_x,w_y,w_z\) should be interpreted as amplitude-envelope radii under this convention. If an intensity-envelope convention is used instead, the usual factor of 2 must be applied consistently.

The transit time can be defined from the envelope duration, for example:

\[
\tau_{\mathrm{transit}}
=
\mathrm{duration}\left[A_{\mathrm{env}}(t)\ge e^{-1}\right].
\]

This event trajectory layer creates the time-dependent signal shape.

## 8. Event-level scattering field time chain

The detector-mode scattering field should be represented in a way that avoids double-counting the Mie phase.

If the normalized detector-mode scattering field is stored as a **complex** quantity:

\[
E_{\mathrm{sca}}(t)
=
A_{\mathrm{env}}(t)
\,f_{\mathrm{coupling}}(t)
\,E_{\mathrm{sca,norm}}^{\mathrm{complex}}
\,e^{i\phi_{\mathrm{path/offset}}(t)}.
\]

If only a **magnitude** is stored, then the phase should be added explicitly:

\[
E_{\mathrm{sca}}(t)
=
A_{\mathrm{env}}(t)
\,f_{\mathrm{coupling}}(t)
\,|E_{\mathrm{sca,norm}}|
\,e^{i\phi_{\mathrm{sca}}(t)}.
\]

A typical phase decomposition is:

\[
\phi_{\mathrm{sca}}(t)
=
\arg\left[\frac{S_2(\theta_{\mathrm{det}})}{k_m}\right]
+
\phi_{\mathrm{path}}(t)
+
\phi_{\mathrm{offset}}.
\]

The path phase can be represented as:

\[
\phi_{\mathrm{path}}(t)=k_m\Delta\ell(t),
\]

or, as a simplified axial surrogate:

\[
\phi_{\mathrm{path}}(t)
\approx
k_m(z(t)-z_{\mathrm{focus}})\cos\theta_{\mathrm{det}}.
\]

Because the current reference phase is not experimentally calibrated, phase and polarity outputs should remain diagnostic.

## 9. NODI interference signal layer

### 9.1 Scalar / joint-overlap working equation

The detector-mode field is:

\[
E_{\mathrm{det}}(t)=E_{\mathrm{ref}}+E_{\mathrm{sca}}(t).
\]

The detector-mode intensity is:

\[
I_{\mathrm{det}}(t)=|E_{\mathrm{det}}(t)|^2.
\]

After blank subtraction:

\[
s_{\mathrm{raw}}(t)
=I_{\mathrm{det}}(t)-|E_{\mathrm{ref}}|^2,
\]

\[
s_{\mathrm{raw}}(t)
=
|E_{\mathrm{sca}}(t)|^2
+
2\operatorname{Re}\left[E_{\mathrm{ref}}E_{\mathrm{sca}}^*(t)\right].
\]

In the weak-scatterer and well-overlapped-reference limit:

\[
|E_{\mathrm{sca}}|^2
\ll
2|E_{\mathrm{ref}}||E_{\mathrm{sca}}|,
\]

so:

\[
s(t)
\approx
2|E_{\mathrm{ref}}||E_{\mathrm{sca}}(t)|\cos\Delta\phi(t),
\]

with:

\[
\Delta\phi(t)=\arg(E_{\mathrm{ref}})-\arg(E_{\mathrm{sca}}(t)).
\]

This is the central formula for interferometric scattering enhancement. It should not be interpreted as a quantitative phase prediction unless reference phase, detector mode, and vector polarization have been calibrated or validated.

For example, if the fields are phase-aligned and:

\[
|E_{\mathrm{ref}}|=100|E_{\mathrm{sca}}|,
\]

then the cross-term amplitude is:

\[
2|E_{\mathrm{ref}}||E_{\mathrm{sca}}|=200|E_{\mathrm{sca}}|^2,
\]

which can be much larger than the pure scattering intensity.

### 9.2 Overlap-factor form

If the reference and scattering angular modes do not fully overlap:

\[
s(t)
=
|E_{\mathrm{sca}}(t)|^2
+
2\operatorname{Re}
\left[
F_{\mathrm{overlap}}E_{\mathrm{ref}}E_{\mathrm{sca}}^*(t)
\right].
\]

Here \(F_{\mathrm{overlap}}\) is a complex mode-overlap factor. It can include mode-matching efficiency and phase mismatch. In a calibrated detector model, \(F_{\mathrm{overlap}}\) should be obtained from a detector-resolved overlap calculation or experimental calibration.

## 10. Particle-channel perturbation diagnostic

Besides Mie scattering, a particle inside the channel may perturb the channel reference field by acting as a local phase object:

\[
E_{\mathrm{ref,total}}(t)
=
E_{\mathrm{ref,blank}}
+
\Delta E_{\mathrm{ref,particle}}(t).
\]

A weak phase-object surrogate can be written as:

\[
\Delta\phi_p(t)
\approx
\frac{2\pi}{\lambda_0}
\left(n_{p,\mathrm{eff}}-n_m\right)L_{\mathrm{eff}}(t).
\]

The corresponding reference perturbation is approximately:

\[
\Delta E_{\mathrm{ref,particle}}(t)
\propto
E_{\mathrm{ref,blank}}
\left(e^{i\Delta\phi_p(t)}-1\right)
\eta_{\mathrm{proj}}(t).
\]

This route must remain diagnostic in the current project because it can double-count physics already represented by Mie forward scattering and particle-induced phase-object scattering. It should not enter the final engineering score as an independent coherent addition unless a double-count guard or full-wave validation is added.

## 11. Noise layer

A generic signal model is:

\[
y(t)=s(t)+b(t)+\epsilon(t),
\]

where:

- \(s(t)\) is the particle signal;
- \(b(t)\) is the baseline/background component;
- \(\epsilon(t)\) is noise.

The current surrogate noise convention includes Gaussian noise, drift, and post-readout noise:

\[
\epsilon(t)\sim \mathcal{N}(0,\sigma^2),
\]

\[
b(t)=b_0+\alpha t.
\]

The current threshold calibration is not empirical blank calibration. Therefore, the model supports relative ranking under a defined noise convention, but it does not support measured false-positive-rate claims.

Future calibrated noise should include:

- empirical blank traces;
- colored noise;
- laser relative intensity noise (RIN);
- detector/electronics gain;
- photon-unit shot noise;
- background leakage and stray light;
- ADC and dynamic-range limits.

## 12. Readout layer: NODI transient / lock-in surrogate

### 12.1 NODI-only EV design readout

A lock-in-like readout can be described conceptually as mixing with a reference waveform and low-pass filtering:

\[
y_{\mathrm{mix}}(t)=s(t)\cos(2\pi f_{\mathrm{lockin}}t+\phi_{\mathrm{ref}}),
\]

\[
y_{\mathrm{lockin}}(t)=\mathrm{LPF}[y_{\mathrm{mix}}(t)].
\]

However, in the current analytic surrogate, the readout should be interpreted mainly as a transit-time / bandpass-envelope response, not as a fully sampled carrier-resolved electronics model.

A useful response factor is:

\[
G_{\mathrm{transit}}=G(\tau_{\mathrm{transit}},\tau_{\mathrm{lockin}}),
\]

with bounds:

\[
G_{\min}\le G_{\mathrm{transit}}\le G_{\max}.
\]

The detected signal can then be represented as:

\[
s_{\mathrm{detect}}(t)
=G_{\mathrm{transit}}\,s_{\mathrm{raw}}(t)+\epsilon_{\mathrm{post}}(t).
\]

Current-run interpretation:

- readout model: lock-in surrogate;
- NODI semantics: bandpass/envelope surrogate;
- internal route: analytic demodulation, no sampled carrier;
- lock-in output units: arbitrary relative units;
- polarity/phase-sensitive classification: not allowed.

### 12.2 POD/NODI paired readout

If a photothermal/POD branch is added later, one may define two coupled readouts:

\[
y_{\mathrm{NODI}}(t)=R_{\mathrm{NODI}}[s_{\mathrm{NODI}}(t)]+c_{\mathrm{POD}\rightarrow\mathrm{NODI}}y_{\mathrm{POD}}(t),
\]

\[
y_{\mathrm{POD}}(t)=R_{\mathrm{POD}}[s_{\mathrm{POD}}(t)]+c_{\mathrm{NODI}\rightarrow\mathrm{POD}}y_{\mathrm{NODI}}(t).
\]

In the current EV/NODI design preset, the POD branch is not a quantitative route.

## 13. Threshold and peak extraction layer

A common threshold rule is:

\[
T=\mu_{\mathrm{bg}}+N_\sigma\sigma_{\mathrm{bg}}.
\]

A robust alternative can use the median and MAD:

\[
T=\mathrm{median}(b)+N_\sigma\cdot 1.4826\cdot \mathrm{MAD}(b).
\]

If using magnitude detection:

\[
s_{\mathrm{peak}}(t)=|s_{\mathrm{detect}}(t)|.
\]

If using positive-pulse detection:

\[
s_{\mathrm{peak}}(t)=s_{\mathrm{detect}}(t).
\]

Peak extraction is then:

\[
\mathrm{peaks}=\mathrm{find\_peaks}(s_{\mathrm{peak}},T,w_{\min},d_{\min}),
\]

where:

- \(w_{\min}\) is the minimum peak width;
- \(d_{\min}\) is the minimum interval between peaks.

For each detected event, the peak-margin statistic is:

\[
\mathrm{peak\_margin}_i
=
\frac{h_i-T}{\sigma_{\mathrm{bg}}}.
\]

The mean peak margin is:

\[
\mathrm{mean\_peak\_margin\_z}
=
\frac{1}{N_{\mathrm{det}}}
\sum_{i=1}^{N_{\mathrm{det}}}\mathrm{peak\_margin}_i,
\]

provided \(N_{\mathrm{det}}>0\). If no event is detected, the statistic must be reported as unavailable or handled by a defined fail value.

## 14. Event QC layer

A detected peak should not automatically become a reliable event. Event QC can be represented schematically as:

\[
QC_i=
\mathbf{1}[\mathrm{width\ in\ range}]
\mathbf{1}[\mathrm{baseline\ acceptable}]
\mathbf{1}[\mathrm{not\ doublet}]
\mathbf{1}[\mathrm{not\ clipped}]
\mathbf{1}[\mathrm{artifact\ risk\ acceptable}]
\cdots
\]

The event QC pass fraction is:

\[
f_{\mathrm{QC}}=\frac{N_{\mathrm{QCpass}}}{N_{\mathrm{events}}}.
\]

An artifact risk score can be represented as:

\[
R_{\mathrm{artifact}}
=w_1R_{\mathrm{baseline}}
+w_2R_{\mathrm{doublet}}
+w_3R_{\mathrm{saturation}}
+w_4R_{\mathrm{template}}
+\cdots.
\]

In the current run, this is an engineering QC/risk surrogate, not a fully calibrated detector dynamic-range or artifact classifier. A high peak is not sufficient; if event QC fails, the case should not be promoted as a robust design.

## 15. Batch statistics layer

For \(N\) simulated events:

\[
\mathrm{detection\_rate}=\frac{N_{\mathrm{det}}}{N},
\]

\[
\mathrm{stable\_detection\_rate}=\frac{N_{\mathrm{stable}}}{N}.
\]

For binomial stability, the Wilson lower bound can be used:

\[
\mathrm{WilsonLB}(k,N)=
\frac{
\hat p+\frac{z^2}{2N}
-z\sqrt{\frac{\hat p(1-\hat p)}{N}+\frac{z^2}{4N^2}}
}{
1+\frac{z^2}{N}
},
\]

with:

\[
\hat p=\frac{k}{N}.
\]

A robust peak-height CV can be defined as:

\[
\mathrm{robust\_CV}
=
\frac{1.4826\cdot\mathrm{MAD}(h_i)}{
\max(|\mathrm{median}(h_i)|,\epsilon)
},
\]

where \(\epsilon\) is a small denominator guard.

The phase-flip fraction can be defined only when detected events exist:

\[
\mathrm{phase\_flip\_fraction}
=
\frac{N_{\mathrm{negative}}}{N_{\mathrm{det}}},
\qquad N_{\mathrm{det}}>0.
\]

If the readout uses magnitude mode, polarity is not a quantitative claim. Positive/negative pulse signs may remain diagnostic only.

## 16. Engineering gate and score layer

An older exploratory score can be represented simply as:

\[
score_{\mathrm{old}}=F(h,\mathrm{rate},CV).
\]

The current engineering-score logic is better summarized schematically as:

\[
score_{\mathrm{eng}}
=F(
 f_{\mathrm{stable}},
 \mathrm{WilsonLB}_{\mathrm{stable}},
 \mathrm{margin}_z,
 \mathrm{SNR}_{\mathrm{local}},
 \mathrm{robustCV},
 \mathrm{phaseFlipRisk},
 R_{\mathrm{artifact}}
).
\]

A linear explanatory form is:

\[
score_{\mathrm{eng}}
\sim
w_1f_{\mathrm{stable}}
+w_2\mathrm{WilsonLB}_{\mathrm{stable}}
+w_3\mathrm{margin}_z
+w_4\mathrm{SNR}_{\mathrm{local}}
-w_5\mathrm{robustCV}
-w_6\mathrm{phaseFlipPenalty}
-w_7\mathrm{artifactPenalty}.
\]

This is not intended as an exact source-code identity unless the actual code weights and normalization functions are cited.

The engineering gate can be represented as:

\[
G_{\mathrm{eng}}
=
\mathbf{1}[N_{\mathrm{det}}\ge N_{\min}]
\mathbf{1}[f_{\mathrm{stable}}\ge f_{\min}]
\mathbf{1}[\mathrm{margin}_z\ge m_{\min}]
\mathbf{1}[QC\ge QC_{\min}]
\cdots.
\]

The final score is:

\[
final\_engineering\_score
=
\begin{cases}
score_{\mathrm{eng}}, & G_{\mathrm{eng}}=1,\\
score_{\mathrm{fail}}, & G_{\mathrm{eng}}=0.
\end{cases}
\]

This prevents a design with high peak height but unstable detection from entering the robust recommendation set.

## 17. EV ensemble aggregation layer

For a design point \(D=(W,H,\lambda)\), the EV ensemble contains multiple EV-like particles:

\[
\mathcal{E}=\{e_1,e_2,\dots,e_M\}.
\]

Each member has outputs such as:

\[
R_j(D),\quad H_j(D),\quad M_j(D),\quad QC_j(D).
\]

A weighted ensemble detection score can be written as:

\[
R_{\mathrm{EV,weighted}}(D)
=
\sum_{j=1}^{M}p_jR_j(D),
\]

where \(p_j\) is a size/RI/population prior. If no reliable prior is available, equal weighting can be used for design-space exploration. The current full-range biomimetic EV profile should be interpreted as an explicit preset ensemble, not as a measured biological population distribution.

Useful robust summaries include:

\[
EV_{p10}(D),\quad EV_{\mathrm{median}}(D),\quad EV_{p90}(D),
\]

and the number of ensemble members that pass hard gates.

## 18. Gold-particle anchor layer

Gold nanoparticles are used as practical scattering anchors. If the anchor is named `Au20` or `gold_20nm`, it should be stated explicitly whether the number refers to **diameter** or **radius**. In the current particle naming convention, `gold_20nm` means a 20 nm diameter gold particle.

A useful relative anchor is:

\[
R_{\mathrm{Au20eq}}(D,p)
=
\frac{A_{\mathrm{signal}}(D,p)}{A_{\mathrm{signal}}(D,\mathrm{Au20\ diameter})}.
\]

Alternatively, if the reference design is fixed:

\[
R_{\mathrm{baseline}}(D,p)
=
\frac{A_{\mathrm{signal}}(D,p)}{A_{\mathrm{signal}}(D_0,\mathrm{Au20\ diameter})}.
\]

This anchor is useful for sanity checks and experiment planning, but it does not replace calibration with blank traces and standard particles.

## 19. Selection function: true population to observed population

The observed population is not equal to the true sample population. A selection function is needed:

\[
p_{\mathrm{obs}}(d,n,\ldots\mid D)
\propto
p_{\mathrm{true}}(d,n,\ldots)
S_{\mathrm{entry}}(d,D)
S_{\mathrm{transport}}(d,D)
S_{\mathrm{survival}}(d,D)
S_{\mathrm{detect}}(d,n,D).
\]

Here:

- \(S_{\mathrm{entry}}\): probability that a particle enters the channel;
- \(S_{\mathrm{transport}}\): transport probability without clogging, adsorption, deformation, or loss;
- \(S_{\mathrm{survival}}\): particle integrity during flow and optical exposure;
- \(S_{\mathrm{detect}}\): optical/electronic detection probability.

The current full-grid event-detection rate is conditional on particles already entering the detection region. It is not a full sample-to-count recovery rate.

## 20. Contaminant overlap and classification layer

For realistic samples, non-EV particles and contaminants may overlap with EV signals. A simplified classifier feature vector may be:

\[
\mathbf{x}_i=
[h_i,w_i,\mathrm{SNR}_i,\lambda\_ratio_i,\mathrm{phase}_i,\mathrm{paired\ response}_i,\ldots].
\]

The classifier or separation rule can be written as:

\[
\hat y_i=f_{\mathrm{class}}(\mathbf{x}_i).
\]

However, without measured control particles and sample-specific contaminants, classification remains a risk-analysis or future-validation layer. It cannot support a claim that a detected biological particle is specifically an exosome rather than a broader EV/EP preparation.

## 21. Count-rate and concentration layer

If concentration and flow are known, the expected arrival rate can be approximated as:

\[
\lambda_{\mathrm{arrive}}=C Q \eta_{\mathrm{entry}},
\]

where:

- \(C\) is number concentration;
- \(Q\) is volumetric flow rate through the relevant detection channel(s);
- \(\eta_{\mathrm{entry}}\) is the entry/transport efficiency.

The expected detected count rate is:

\[
\lambda_{\mathrm{det}}=\lambda_{\mathrm{arrive}}p_{\mathrm{det}}.
\]

For a time window \(T\), a Poisson count model would be:

\[
N_{\mathrm{det}}\sim \mathrm{Poisson}(\lambda_{\mathrm{det}}T).
\]

The current project does not apply a full count-prediction model. Current detection rates are conditional per-event detectability metrics, not direct concentration readouts.

## 22. POD / photothermal branch

The photothermal/POD branch is currently not a quantitative active route, but conceptual formulas can be written for future extension.

### 22.1 Refractive-index sensitivity

A photothermal signal can be treated as a temperature-induced refractive-index perturbation:

\[
\Delta n(\mathbf{r},t)=\frac{dn}{dT}\Delta T(\mathbf{r},t).
\]

The optical field perturbation is then related to:

\[
\Delta E_{\mathrm{POD}}
\propto
\int
\frac{\partial E_{\mathrm{det}}}{\partial n(\mathbf{r})}
\Delta n(\mathbf{r},t)
d\mathbf{r}.
\]

### 22.2 Heat source

The absorbed power of a particle is:

\[
P_{\mathrm{abs}}=I_{\mathrm{exc}}C_{\mathrm{abs}}.
\]

A heat equation model would be:

\[
\rho C_p\frac{\partial T}{\partial t}
=
\nabla\cdot(\kappa_{\mathrm{th}}\nabla T)+Q_{\mathrm{heat}}(\mathbf{r},t).
\]

Here \(\kappa_{\mathrm{th}}\) is thermal conductivity. It is not the optical wave number \(k_m\).

The current project marks this branch as unavailable because excitation wavelength, excitation power, heat-diffusion model, and POD detector calibration are not yet configured.

## 23. Claim governance layer

The model must separate what is allowed from what is blocked.

### Allowed under the current project status

- Relative ranking of channel width, depth, and wavelength under the same model settings.
- Within-lane comparison of designs.
- Mechanistic interpretation of why reference field, scattering field, and interference term change with wavelength and geometry.
- Candidate-chip matrix generation for first-round experiments.
- Identification of weak-reference, high-exposure, calibration-missing, interface-correction-missing, or high-NA scalar-risk cases.

### Blocked until calibration or full-wave validation

- Absolute detector voltage.
- Absolute photon-unit SNR.
- Absolute LOD.
- Absolute concentration or sample particle count.
- Quantitative material classification from phase/polarity.
- Quantitative claim that a detected biological particle is specifically an exosome rather than a broader EV/EP preparation.
- Quantitative interface-corrected scattering near glass/water boundaries.
- Quantitative full-vector high-NA optical phase prediction.
- Quantitative pressure-flow recovery without measured pressure-flow traces and complete external network geometry.

A useful eligibility split is:

```text
relative_design_eligible
within_lambda_design_eligible
absolute_global_green_eligible
final_green_eligible
```

The present model mainly supports `relative_design_eligible` and engineering ranking. Absolute green claims require calibration gates.

## 24. Final design score and candidate-chip matrix

The final design decision should not be a single maximum-score channel. It should be a candidate matrix.

A multi-objective design score can be written conceptually as:

\[
DesignScore(D)
=
w_1EV_{p10}(D)
+w_2EV_{\mathrm{median}}(D)
+w_3R_{\mathrm{Au20}}(D)
+w_4ReferenceBalance(D)
+w_5QC(D)
\]

\[
-w_6FluidicRisk(D)
-w_7SelectionBias(D)
-w_8ContaminantOverlap(D)
-w_9DetectorDisagreement(D)
-w_{10}ExposureRisk(D).
\]

This formula is a design-governance expression, not necessarily the exact source-code scoring formula.

Hard blockers cannot be compensated by a high score:

\[
Eligible_{\mathrm{green}}
=
G_{\mathrm{unit}}
\land G_{\mathrm{readout}}
\land G_{\mathrm{detector}}
\land G_{\mathrm{doublecount}}
\land G_{\mathrm{QC}}
\land G_{\mathrm{exposure}}
\land G_{\mathrm{claim}}.
\]

If:

\[
Eligible_{\mathrm{green}}=0,
\]

then the design remains exploratory/yellow even if the numerical score is high.

Recommended output should be a chip-and-control panel, for example:

```text
balanced_primary
aggressive_high_sensitivity
fluidically_robust
tsuyama_like_control
weak_reference_control
short_wavelength_candidate
long_wavelength_control
low_contaminant_overlap_control
```

This is more scientifically honest than choosing a single “best” channel.

## Final one-sentence statement

The current EV/NODI formula chain is scientifically coherent and matches the project as a **relative, governed, engineering-surrogate simulator for first-round channel and wavelength selection**, while absolute optical, detector, concentration, pressure-flow, full-vector phase, interface-corrected, and biological-specificity claims remain explicitly blocked until calibration and validation data are added.
