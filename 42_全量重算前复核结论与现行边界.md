## 现在可以重算的理由

1. 旧结果库已经与 current code / current schema 分离，不能继续作为标准真值。
2. 当前代码已经把 paper-aligned、calibrated、engineering surrogate 三类口径拆开。
3. 无实测数据时，模板与 synthetic fixture 只暴露字段和单位，不会解锁 calibrated claims。
4. 全量重算现在可以生成 current governed engineering / EV design decision library，不需要等待未来实验闭环；但输出必须保持 relative/proxy/diagnostic claim level。

## 当前主边界

当前结果允许解释为：

- engineering ranking
- paper-aligned comparison lane
- reference/scattering/detector/readout/count 等分 lane provenance
- calibration-ready schema and templates
- P0/P1 EV design decision blockers, smoke fields, and dashboard visibility

当前结果不允许解释为：

- photon-unit calibrated NODI absolute amplitude
- experimental count-rate prediction
- measured reference phase / absolute polarity claim
- quantitative POD thermal signal
- full-wave near-interface electromagnetic closure
- calibrated SNR / absolute LOD / absolute EV concentration / biological EV specificity
