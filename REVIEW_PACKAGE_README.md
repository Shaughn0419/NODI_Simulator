# EV/NODI Post-v2 Relative Audit Package

This P0 release package supports a no-measured-data relative audit. It records reproducible paths, hashes, scaffold roles, paper provenance, claim-language blockers, and P0b route adjudication artifacts for review.

v2 sidecars provide BFP ROI/Jacobian, Tsuyama BFP reference, Mie-to-power guardrail, noise/readout scenario, selected-annulus, and EV/sample uncertainty no-measured-data audit layers; however, the v1 full-grid main library remains a theta/phi surrogate, pupil/slit surrogate, arbitrary-unit relative engineering library, so all conclusions are relative candidate-audit conclusions and not calibrated physical predictions.
